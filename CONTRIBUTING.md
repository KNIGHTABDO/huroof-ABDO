# Contributing to حروف مع عبدو (Huroof Game)

First off, thank you for considering contributing to **حروف مع عبدو**! It's people like you that make this community such a great place to build amazing projects.

## Code of Conduct
By participating and contributing to this project, you agree to uphold our inclusive and respectful environment. Please be professional, kind, and collaborative in all interactions.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please create a new issue and include:
- A clear, descriptive title.
- Steps to reproduce the issue.
- Expected behavior vs. actual behavior.
- Screenshots, if applicable.
- Your OS, browser, and environment details.

### Suggesting Enhancements
Have a great idea to improve the game, add new features, or refine the design? Open an issue tagged as `enhancement`. Describe your brilliant idea in detail and how it benefits the users.

### Pull Requests
Ready to write some code? 
1. **Fork** the repository and clone it locally.
2. **Branch** out from `main` (`git checkout -b feature/amazing-feature-name`).
3. **Commit** your changes with clear, descriptive commit messages.
4. **Push** your branch (`git push origin feature/amazing-feature-name`).
5. **Open a Pull Request** describing precisely what changes you made and why.

## Development Style Guide
- **Arabic First:** We heavily rely on the RTL format and Arabic typography (Cairo font). Ensure new components respect RTL layout rules.
- **Design Aesthetic:** Keep the Apple-style aesthetic in mind. Use blur, smooth transitions, high-contrast dark mode colors, and sufficient padding.
- **Code Quality:** Ensure there are no warnings or errors during `npm run lint`. Maintain clean, modular React components.

Thank you for your time and expertise!
